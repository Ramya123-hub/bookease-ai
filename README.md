# 🌐 BookEase AI

BookEase AI is a modern AI-powered service booking platform that connects customers with service providers in a simple and efficient way. Users can browse services, book appointments, and manage their schedules, while providers can list services and manage bookings through dedicated dashboards.

---

## 🚀 Features

### 👤 Customer Features
- Sign up / Login system (frontend auth)
- Browse available services
- Book appointments
- View and manage bookings
- Cancel appointments

### 🧑‍💼 Provider Features
- Provider dashboard
- Add and manage services
- View customer bookings
- Accept / reject appointments
- Manage business profile

### ⚙️ General Features
- Role-based system (Customer / Provider)
- Clean and responsive UI (React + Tailwind)
- Frontend-only authentication using localStorage
- Smooth navigation using React Router

---

## 🛠️ Tech Stack

- React.js

- Tailwind CSS
- LocalStorage (for authentication & data)
